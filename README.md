# Review Design Guidelines

This is a code bundle for Review Design Guidelines. The original project is available at https://www.figma.com/design/churtR1pAOwO5qeDH5zkZr/Review-Design-Guidelines.

## Running the code

Run `npm i` to install dependencies.

Run `npm run dev` to start the Next.js development server.

Run `npm run build` and `npm start` for production.
